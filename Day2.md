<h2>Data Types:</h2>

<h3>Primitive data types:</h3>

<h4>String</h4>
represented by these:

```javascript
'Ahmad'
"Ahmad"
`Ahmad`
```

<h4>Number</h4>
all of these are numbers:

```javascript
9
525600
-3.45
1.21e9
Infinity
```

<h4>boolean</h4>

```javascript
true
false
```

<h4>undefined</h4>
It represents the accidental nothing.

<h4>null</h4>
It represents the deliberate nothing.
