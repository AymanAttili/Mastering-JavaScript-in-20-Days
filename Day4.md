<h2>Functions</h2>

- values are things, variables point to things and **functions do things**.

<h3>Declaring (creating) a function:</h3>

    ```javascript
    function mulByFive(x){
        return x * 5;
    }
    ```

<h3>Calling (using) a function:</h3>

```javascript
const ten = mulByFive(2);
```
